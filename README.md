# 📈 数字教育视域下高质量知识传播的传播机制与治理创新 (Bilibili-Edu-Analysis)

> **2026年全国大学生统计建模大赛 参赛项目**
> 
> 本项目基于B站学习生态的多源数据，创新性融合自然语言处理（NLP）与经典结构方程模型（SEM），对网络学习空间中的“过度营销”与“流量留存”机制进行深度实证分析。

## 💡 项目背景
当前网络学习空间普遍面临“流量至上”与“过度包装”的治理挑战。高质量硬核知识常被“降智/保姆级”等营销标签掩盖。本项目旨在通过跨学科的统计建模，剥离表层包装的干扰，科学测度实质深层内容的传播机制，为平台算法重构与数字教育治理提供量化的数据支撑。

## 🛠️ 技术栈与工具
* **数据采集与清洗**：Python (Requests, BeautifulSoup, Pandas) / 八爪鱼
* **NLP 与特征工程**：SnowNLP (情感极性计算), scikit-learn (LDA 主题模型信息熵), 领域词典匹配
* **统计建模分析**：SPSS (分层回归, K-Means聚类), Amos (多群组结构方程模型 SEM)

## 📂 核心研究路径 (Pipeline)
1. **多源数据采集**：构建B站考研、考公、自学三大场景数千条异质性视频样本库。
2. **非结构化文本量化**：提取内容专业度、情感唤醒度与话题聚焦度三大核心实质特征。
3. **用户粘性指数构建**：基于 Log 播放量与修正交互比率，客观赋权深度粘性。
4. **机制检验与建模**：利用分层回归剥离浅层包装因素；利用 Amos 路径分析进行流量留存悖论检验。

## 📊 主要科学发现
1. **流量留存悖论 (Traffic Retention Paradox)**：证实表层营销标签虽能带来初期的高曝光，但在深度交互转化上呈显著的负向衰减。
2. **过度营销反噬效应**：过高的情感唤醒度与贩卖焦虑文案，对长时程、高门槛的知识内容传播具有破坏性作用。
3. **策略输出**：为平台重构以“时长、专业度、深度粘性”为核心的分发算法提供了机制证明与数据参考。

## 🚀 仓库结构说明
- `/code`：包含数据爬取、NLP 特征提取以及统计建模的核心代码与脚本。
- `/data`：存放用于复现分析的脱敏抽样数据集。
- `/docs`：包含项目完整的实证研究报告 PDF 版本及相关图表资源。
- `/results`：包含 Amos 路径拟合图及分层回归结果输出。
